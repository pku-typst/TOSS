# Typst reference data

The generated `api-0.15.0.json` and `api-0.15.0-bm25.json` files are pinned
copies of the normalized Typst API index from
[`lucifer1004/claude-skill-typst`](https://github.com/lucifer1004/claude-skill-typst).
That project is licensed under the MIT License included in this directory.

The API descriptions are derived from the Typst standard library and its
documentation, which are distributed under the Apache License 2.0. A copy is
available at [`third-party/typst.ts/LICENSE`](../../../../third-party/typst.ts/LICENSE).

The exact source revision, paths, and checksums are recorded in
`manifest.json`. Run `npm run sync:typst-docs` only when deliberately updating
the pinned reference data.
