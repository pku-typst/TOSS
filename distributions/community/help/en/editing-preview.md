## Edit source

Select a text file in **Files** and edit it in the center pane. Typst syntax highlighting, undo/redo, search, and standard operating-system shortcuts are provided by the editor.

Remote collaborator changes are merged into the active document. Presence indicators show who is currently connected.

## Read the preview

The default canvas renderer is optimized for the live editing loop. Use the preview toolbar to fit the whole page, fit the page width, zoom, or generate and download a PDF in the browser.

Some project types may also expose an explicit background-build control after sign-in. That durable task uses a fixed server-accepted snapshot and appears in **Tasks**; it does not replace the live browser preview or automatically include edits made after submission.

Clicking supported source content can reveal the corresponding preview location. Clicking mapped preview content can return to its source position.

## Resolve errors

Compilation diagnostics identify the file and source location when available. Fix the first error before chasing later messages: one syntax error can produce several downstream diagnostics.

Package and compiler downloads are cached by the browser. The first compilation on a device is therefore slower than later edits.
