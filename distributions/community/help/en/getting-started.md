## Start with a template

Open **Gallery** to create an independent project from a maintained template, or create a blank project from **Projects**. A copied template is yours to edit; later template updates do not rewrite the project.

## Understand the workspace

- **Files** contains Typst source, data, images, fonts, and other assets.
- **Editor** changes the selected text file.
- **Preview** compiles the project in your browser.
- **Tasks** (after sign-in) shows durable builds and conversions that continue after the current tab closes.
- **Settings** contains sharing, project type, export, and Git controls.
- **Revisions** lets you inspect earlier checkpoints.

Edits are synchronized to the service automatically. `Ctrl+S` on Windows/Linux or `Cmd+S` on macOS requests an immediate save and does not open the browser save dialog.

## Keep important work portable

Platform storage policy depends on the deployment. Download a project archive or connect Git when a project needs an independent, durable copy.
