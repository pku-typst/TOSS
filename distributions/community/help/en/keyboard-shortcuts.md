## Editor shortcuts

`Mod` means `Cmd` on macOS and `Ctrl` on Windows/Linux.

| Action | Shortcut |
| --- | --- |
| Request an immediate save | `Mod+S` |
| Select all | `Mod+A` |
| Find in the current file | `Mod+F` |
| Undo | `Mod+Z` |
| Redo | `Shift+Mod+Z` |
| Indent selection | `Tab` |
| Unindent selection | `Shift+Tab` |

Standard browser shortcuts still apply outside the editor. When a dialog or project-title editor is open, `Escape` cancels the current interaction.
