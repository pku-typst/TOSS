## 共享项目

项目所有者可以在 **设置 → 访问权限** 中管理协作。长期协作建议使用具名成员；共享链接适合具体的评审或编辑场景，并可由所有者随时撤销。

只读权限不能修改项目；写权限可以编辑项目文件；管理权限可以控制共享、模板和仓库集成。

## 应对网络中断

短暂断网后编辑器会自动重连。在连接状态恢复在线前请保持标签页打开。如果服务长时间不可用，不要假定尚未发送的修改已经到达服务端。

## 保留长期副本

协作服务与 Git 是两套独立存储。部署提供 Git 集成时，请连接自己控制的仓库并主动同步；也可以随时下载项目归档。

在把 Git 仓库视为最新副本前，请检查仓库状态和最后同步时间。
